# 680b2ab5-security-audit-compliance-chec
REST API that performs automated compliance checks against major security frameworks. Input your system configuration, infrastructure specs, and policy documents to receive compliance gap analysis for SOC 2 Type II, ISO 27001, NIST CSF, CIS Controls, PCI DSS Level 1, HIPAA Security Rule, and FedRAMP Moderate. Returns prioritized remediation roadmap
